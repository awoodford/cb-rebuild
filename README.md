## Project Set-up

### Prerequisites:

_Node Version:_ v10.19.0

### Install NPM packages

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lint's and fixes files

```
npm run lint
```
